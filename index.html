Indrodução


Está começando do zero?

Instalações necessárias:

  • Ruby
  • NodeJS
  • GruntJS
  • Compass

Iniciando o desenvolvimento:

Para fazer qualquer alteração de CSS é necessário deixar o GruntJs rodando. Para executa-lo, abra o prompt de comando o Ruby.

Existem 3 task do Grunt para executar no prompt de comando:

  • grunt default (ou grunt): Esta task roda as duas tarefas abaixo (grunt dev e grunt prod), para padronizar todos os arquivos CSS para o ambiente de desenvolvimento e o ambiente de produção;
  • grunt dev: Esta task, assim que executada, fica observando todos os arquivos *.scss para compila-los automaticamente;
  • grunt prod: Esta task, assim que execudada, gera os arquivo css minificados, para ser usado em ambiente de produção.

Padroes CSS

Abaixo estão listados a padronização de escrita de CSS.

Aspas

Utilizar sempre aspas duplas (") no CSS.

					.clearfix:after { content: " "; }

					input[type="button"] { ... }

Cores

Para declarar a cor hexadecimal no CSS use abreviações quando possível (em caso de repedições), assim:

					/* Recomendado */
					p { color: #fff; }

					/* Não Recomendado */
					p { color: #FFFFFF; }
					p { color: #ffffff; }

Comentários

Errado

					p > /*span >*/ span {color: #fff; }
					p { color: #000; } /* Errado */
					p { color: #000; /* Errado */ }
